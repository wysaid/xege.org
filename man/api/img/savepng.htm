功能：

将图像以 PNG 格式保存到文件中。(如果文件已存在，会覆盖原文件。)

声明：

int savepng(
    PCIMAGE  pimg,
    const char*   filename,
    bool     withAlphaChannel = false
);

int savepng(
    PCIMAGE  pimg,
    const wchar_t*  filename,
    bool     withAlphaChannel = false
);


参数：
pimg
要保存的图像

filename
保存的图像文件名

withAlphaChannel
是否保存图像透明度通道，true: 保存, false: 不保存。默认为不保存。

返回值：

成功返回0，否则返回非0，若pimg传入错误，会引发运行时异常。
