功能：

往目标路径中添加一个路径。

声明：

void ege_path_addpath(
    ege_path* dstPath,          // 目标路径
    const ege_path* srcPath,    // 源路径
    bool connect                // 是否连接两个路径(非闭合时)
);


参数：

dstPath
目标路径

srcPath
源路径

connect
是否将目标路径末尾与源路径起点相连。
·如果为 true，则会在目标路径的最后一个图形单元和源路径的第一个图形单元
都是非闭合时，用直线将目标路径末尾点与源路径起始点进行连接。
·如果为 false, 则不连接。

返回值：

（无）
