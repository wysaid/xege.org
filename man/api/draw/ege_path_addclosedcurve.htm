主页 / 库函数目录 / 图形绘制 / ege_path_addpolygon

功能：

往路径中添加一个闭合的基数样条曲线。

说明：
添加的图形为闭合图形，相当于开始一个新的图形单元然后添加图形并将其闭合。
不会自动与上一个图形单元进行连接，也不会闭合上一个图形单元。

声明：

void ege_path_addpolygon(
    ege_path* path,           // 路径
    int numOfPoints,          // 控制点个数
    const ege_point* points   // 控制点数组
);

void ege_path_addpolygon(
    ege_path* path,           // 路径
    int numOfPoints,          // 控制点个数
    const ege_point* points,  // 控制点数组
    float tension             // 曲线张力
);


参数：

path
路径

numOfPoints
控制点顶点个数，至少为 3

points
控制点顶点数组

tension
曲线张力，值为 0 时曲线变为直线

返回值：

（无）
