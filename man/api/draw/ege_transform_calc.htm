主页 / 库函数目录 / 图形绘制 / ege_transform_translate

功能：

这个函数计算在当前设置的坐标变换下，一个点经过变换后得到的实际坐标。

声明：


ege_point ege_transform_calc(
    float x,              // 点的 x 坐标
    float y,              // 点的 y 坐标
    PCIMAGE pimg = NULL  //目标图像指针，NULL表示绘制到屏幕
);

ege_point ege_transform_calc(
    ege_point  point,    // 点的坐标
    PCIMAGE pimg = NULL  //目标图像指针，NULL表示绘制到屏幕
);


参数：

point
点的坐标

x, y
点的 x, y 坐标

返回值：
经过变换后的实际坐标
