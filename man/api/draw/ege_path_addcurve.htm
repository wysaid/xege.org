主页 / 库函数目录 / 图形绘制 / ege_path_addcurve

功能：

往路径当前图形单元中添加一个基数样条曲线。

说明：
    如果路径中的当前图形单元没有闭合，会自动在当前图形单元的末尾点与新添加
图形的起始点之间用一条直线连接。
    如果不希望自动连接，可以在添加前调用 ege_path_start 开始新的图形单元

声明：

void ege_path_addcurve(
    ege_path* path,           // 路径
    int numOfPoints,          // 控制点个数
    const ege_point* points   // 控制点数组
);

void ege_path_addcurve(
    ege_path* path,           // 路径
    int numOfPoints,          // 控制点个数
    const ege_point* points,  // 控制点数组
    float tension             // 曲线张力
);


参数：

path
路径

numOfPoints
控制点顶点个数，至少为 2

points
控制点顶点数组

tension
曲线张力，值为 0 时曲线变为直线

返回值：

（无）
