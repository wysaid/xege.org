主页 / 库函数目录 / 图形绘制 / ege_fillpath

功能：

    绘制一个填充路径(无边框)，填充颜色由 setfillcolor() 指定。

说明：
    如果路径没有闭合，则绘制出的填充图形相当于路径闭合(首尾用直线连接)后形成的图形
    如果路径边框存在交叉，则存在一个该如何判断哪些部分是区域内部的问题。
    ege_path_setfillmode() 可以设置两种不同的路径填充模式，分别采用不同的方式来
进行判断，得到的图形可能并不相同。

声明：

void ege_fillpath(
    const ege_path* path,  // 路径
    PCIMAGE pimg = NULL    // 目标图像，NULL 表示绘制到窗口
);

void ege_fillpath(
    const ege_path* path,  // 路径
    int x,                 // 目标位置的 x 坐标
    int y,                 // 目标位置的 y 坐标
    PCIMAGE pimg = NULL    // 目标图像，NULL 表示绘制到窗口
);


参数：

path
路径

x
目标位置的 x 坐标
y
目标位置的 y 坐标

pimg
目标图像（NULL 为窗口）

返回值：

（无）
