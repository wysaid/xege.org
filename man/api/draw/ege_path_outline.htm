功能：

转换路径上的控制点，使其仅表示路径的轮廓。

说明：
同时也会平展路径，将路径上的曲线变为近似的直线段。

声明：

void ege_path_outline(
    ege_path* path,                             // 路径
    const ege_transform_matrix* matrix = NULL   // 变换矩阵
);
void ege_path_outline(
    ege_path* path,                             // 路径
    const ege_transform_matrix* matrix,         // 变换矩阵
    float flatness                              // 平整度
);


参数：

path
路径的指针

matrix
变换矩阵，指定要对路径上的控制点进行的变换。
如果为 NULL 则不进行变换。

flatness
平整度。指定路径与其平展近似值之间的最大误差。 减少平整度会增加近似值中的线段数。

返回值：

（无）
