功能：

销毁路径

说明：
    通过 ege_path_create(), ege_path_createfrom() 和 ege_path_clone()创建
的路径为动态内存分配，因此当路径不再使用时，需要调用 ege_path_destroy() 进
行销毁，以免发生内存泄漏。

声明：

void ege_path_destroy(
    const ege_path* path           // 路径
);


参数：

path
路径的指针

返回值：

（无）

示例：

ege_path* path = ege_path_create();        // 创建路径 
ege_drawpath(path);                        // 绘制路径
ege_path_destroy(path);                    // 销毁路径
