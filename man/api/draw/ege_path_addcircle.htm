主页 / 库函数目录 / 图形绘制 / ege_path_addcircle

功能：

往路径中添加一个圆。

说明：
添加的图形为闭合图形，相当于开始一个新的图形单元然后添加图形并将其闭合。
不会自动与上一个图形单元进行连接，也不会闭合上一个图形单元。

声明：

void ege_path_addcircle(
    ege_path* path,        // 路径
    float x,               // 圆心 x 坐标
    float y,               // 圆心 y 坐标
    float radius           // 圆的半径
);


参数：

path
路径

x, y
圆心坐标

radius
圆的半径

返回值：

（无）
