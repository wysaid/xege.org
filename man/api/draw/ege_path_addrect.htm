主页 / 库函数目录 / 图形绘制 / ege_path_addrect

功能：

往路径中添加一个矩形。

说明：
添加的图形为闭合图形，相当于开始一个新的图形单元然后添加图形并将其闭合。
不会自动与上一个图形单元进行连接，也不会闭合上一个图形单元。

声明：

void ege_path_addrect(
    ege_path* path,        // 路径
    float x,               // 矩形左上角 x 坐标
    float y,               // 矩形左上角 y 坐标
    float width,           // 矩形宽度
    float height           // 矩形高度
);


参数：

path
路径

x, y
矩形左上角坐标

width
矩形的宽度

height
矩形的高度

返回值：

（无）
