功能：

获取路径上所有控制点的坐标。

说明：


声明：

ege_point* ege_path_getpathpoints(
    const ege_path* path,              // 路径
    ege_point* points = NULL           // 用于存储路径控制点坐标的缓冲区首地址
);


参数：

path
路径的指针

points
用于存储路径控制点坐标的缓冲区首地址。需要能容纳所有控制点坐标，缓冲区创建前可通过
ege_path_pointcount 获取路径控制点个数。
如果参数为 NULL，则函数会通过动态内存分配创建一个控制点坐标数组，函数返回值即
为该数组首地址，不再使用时需要手动释放内存。

返回值：

存储路径控制点坐标的缓冲区首地址。如果 points 参数为 NULL，则缓冲区为函数创建，
后续需要手动释放内存。
