主页 / 库函数目录 / 图形绘制 / ege_path_addarc

功能：

往路径当前图形单元中添加一条弧。

说明：
    如果路径中的当前图形单元没有闭合，会自动在当前图形单元的末尾点与新添加
图形的起始点之间用一条直线连接。
    如果不希望自动连接，可以在添加前调用 ege_path_start 开始新的图形单元

声明：

void ege_path_addarc(
    ege_path* path,           // 路径
    float x,                 // 包围矩形左上角 x 坐标
    float y,                 // 包围矩形左上角 y 坐标
    float width,             // 包围矩形宽度
    float height,            // 包围矩形高度
    float startAngle,        // 起始角(单位为角度)
    float sweepAngle         // 扫描角(顺时针方向，单位为角度)
);


参数：

path
路径

x
包围矩形左上角 x 坐标

y
包围矩形左上角 y 坐标

width
包围矩形宽度

height
包围矩形高度

startAngle
起始角(单位为角度)

sweepAngle
扫描角(顺时针，单位为角度)

返回值：

（无）
