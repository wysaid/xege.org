主页 / 库函数目录 / 图形绘制 / ege_path_createfrom

功能：

从控制点数据中创建路径，并返回路径的指针。

说明：
路径的控制点数据可由 ege_path_getpathpoints, ege_path_getpathtypes
和 ege_path_pointcount 获取

说明：
通过 ege_path_createfrom() 创建的路径为动态内存分配，因此当路径不再使用时，需要
调用 ege_path_destroy() 进行销毁，以免发生内存泄漏。

声明：

ege_path* ege_path_createfrom(
    const ege_point* points,      // 控制点坐标数组
    const unsigned char* types,   // 控制点类型数组
    int count                     // 控制点个数
);


参数：

points
控制点坐标数组

types
控制点类型数组

count
控制点个数

返回值：

创建的路径的指针。
当路径不再使用时，需要调用 ege_path_destroy() 进行销毁，以免发生内存泄漏。
