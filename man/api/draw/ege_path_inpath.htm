功能：

判断点是否在路径填充区域内部。

声明：

void ege_path_inpath(
    ege_path* path,         // 路径
    float x,                // 点的 x 坐标
    float y                 // 点的 y 坐标
);

void ege_path_inpath(
    ege_path* path,         // 路径
    float x,                // 点的 x 坐标
    float y,                // 点的 y 坐标
    PCIMAGE pimg            // 目标图像
);


参数：

path
路径的指针

x, y
点的坐标

pimg
目标图像。路径填充区域受所设置的路径填充模式影响，因此为图像设置的不同填充
模式可能会得到不同的结果。

返回值：

（无）
