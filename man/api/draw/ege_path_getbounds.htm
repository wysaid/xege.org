功能：

获取路径的边界（包围矩形）。
获取的边界将足够大，足以将路径括起来。

说明：


声明：

ege_rect ege_path_getbounds(
    const ege_path* path,                       // 路径
    const ege_transform_matrix* matrix = NULL   // 变换矩阵
);
ege_rect ege_path_getbounds(
    const ege_path* path,                       // 路径
    const ege_transform_matrix* matrix,         // 变换矩阵
    PCIMAGE pimg                                // 目标图像
);


参数：

path
路径的指针

matrix
变换矩阵，计算边界前先对路径进行坐标变换，变换后再进行计算(变换仅用于此次计算，不更改源路径)。
如果为 NULL 则不进行变换。

pimg
目标图像。
线条线宽和线条连接处样式等设置的不同会影响边界的大小。

返回值：

路径的边界（矩形）。
