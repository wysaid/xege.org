主页 / 库函数目录 / 图形绘制 / ege_path_setfillmode

功能：

设置路径的填充模式。

说明：
不同的填充模式决定如何确定一个点是否在区域内部，如果
路径发生交叉、包含等，使用不同的填充模式可能会得到不同的填充结果。

声明：

ege_point ege_path_setfillmode(
    ege_path* path,      // 路径
    fill_mode mode       // 填充模式
);


参数：

path
路径的指针

mode
填充模式，可以设置以下其中一个值：


    
枚举值含义

    
FILLMODE_DEFAULT(默认)       0默认，绘制时与图像设置一致，未指定图像时
为 FILLMODE_ALTERNATE 模式

    
FILLMODE_ALTERNATE     1奇偶交替模式

    
FILLMODE_WINDING  2缠绕模式




返回值：

（无）
